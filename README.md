# Machine Telemetry Analysis using Tableau

This project analyzes machine telemetry data collected from four Daikibo factories to identify operational downtime and machine failure patterns.

## Objective
Identify:
- Which factory experienced the highest downtime
- Which machines contributed most to the downtime

## Dataset
Telemetry data collected every 10 minutes from machines in:

- Tokyo
- Osaka
- Berlin
- Shenzhen

Each record contains:
- Factory
- Device Type
- Machine Status
- Timestamp

## Tools Used
- Tableau
- JSON dataset
- Data visualization
- Dashboard design

## Dashboard
The dashboard includes:

- Downtime per factory
- Downtime per device type
- Interactive filtering

## Key Insight
Seiko Factory (Osaka) had the highest downtime.

LaserWelder machines contributed most to failures.

## Business Impact
This analysis helps organizations:
- Improve predictive maintenance
- Reduce operational downtime
- Optimize factory performance
