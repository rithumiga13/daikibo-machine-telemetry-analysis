# Dataset Information

The dataset used in this project contains machine telemetry data from Daikibo factories.

Due to GitHub file size limits, the dataset is not uploaded directly to this repository.

## Dataset Description
Telemetry messages collected every 10 minutes from machines in 4 factories:

- Tokyo, Japan
- Osaka, Japan
- Berlin, Germany
- Shenzhen, China

Fields included:
- Factory
- Device Type
- Status
- Timestamp

## Source
Provided as part of the Deloitte Data Analytics Virtual Experience Program (Forage).

To reproduce the analysis, download the dataset from the simulation resources.
