Analyzed machine telemetry data from 4 global factories to identify operational downtime patterns.


Processed JSON telemetry data with machine status updates every 10 minutes using Tableau.


Designed a calculated metric “Unhealthy” to quantify machine downtime from status logs.


Built interactive Tableau dashboards visualizing downtime by factory and device type.


Implemented dashboard filtering to enable dynamic analysis of machine failures by location.


Identified Seiko Factory (Osaka) as the highest downtime location, with LaserWelder machines contributing most failures.


Generated insights supporting predictive maintenance and operational optimization.
